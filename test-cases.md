# Test Cases – Zomato Landing Page Clone

---

## TC-01: Page Load Verification
**Description:** Verify that the landing page loads successfully.  
**Precondition:** User has a modern web browser.

**Steps:**
1. Open the browser.
2. Open `index.html`.

**Expected Result:**  
Page loads without errors and displays all sections correctly.

---
